Descripción del Proyecto

Explora el montaje del equipo pasando por cada punto para ver los componentes destacados.

"Diseño mural retro con los componentes distribuidos como una pieza expositiva interactiva."

El enfoque principal de este proyecto es combinar una presentacion visual llamativa con una lectura clara de cada zona importante del montaje.
